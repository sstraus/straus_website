Personal portfolio website simulating a Mac terminal running `straus-code` CLI.

**Live:** [straus.it](https://straus.it)

## Features

- Mac terminal window simulation with traffic light buttons
- ASCII art portrait with hover-to-image effect
- Markdown-based content
- Fake zsh commands for fun
- Mobile responsive

## Commands

```
help     - Show available commands
about    - About me
blog     - List blog posts
read     - Read a blog post
skills   - Technical skills
contact  - Contact information
clear    - Clear terminal
```

## Tech Stack

Vanilla JavaScript, ES6 modules, no frameworks.
